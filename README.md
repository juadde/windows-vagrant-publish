# Windows-11-fr-input build

This repository automates the delivery of the [Kppqju77/windows-11-fr-input](https://portal.cloud.hashicorp.com/vagrant/discover/kppqju77/windows-11-fr-input) Vagrant box.

It is based on [rgl/windows-vagrant](https://github.com/rgl/windows-vagrant) and uses the same build process.

The only difference is that the locales are set to fr-FR.
